阿屜報氣象
==========

臺大學生會氣象 API，由[第 28 屆臺大學生會福利部](https://www.facebook.com/NTUWelfare)設計、[臺灣大學大氣系](http://www.as.ntu.edu.tw)資料提供。

可以在[這裡](http://weather.ntustudents.org)看到實況，也可以在[這裡](http://weather.ntustudents.org/api)使用 API。
